IIFEs ahora Block
problemas en bloques

Un uso común de las expresiones de función Inmediatamente invocados es encerrar los valores dentro de su alcance. En ES6, ahora tenemos la capacidad de crear ámbitos basados en bloques y por lo tanto no se limitan exclusivamente al ámbito basado en la función.


(function () {
    var food = 'Meow Mix';
}());

console.log(food); // Reference Error

Usando ES6 block: 


{
    let food = 'Meow Mix';
};

console.log(food); // Reference Error

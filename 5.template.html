Template literals
facil uso de strings

El uso de plantillas literales , ahora podemos construir cadenas que tienen caracteres especiales en ellas sin necesidad de escapar de ellos de forma explícita.


var text = "This string contains \"double quotes\" which are escaped.";
let text = `This string contains "double quotes" which don't need to be escaped anymore.`;

Los Templates Literal también apoyan la interpolación, lo que hace que la tarea de la concatenación de cadenas y valores: 


var name = 'Tiger';
var age = 13;

console.log('My cat is named ' + name + ' and is ' + age + ' years old.');

Mucho más simple:


const name = 'Tiger';
const age = 13;

console.log(`My cat is named ${name} and is ${age} years old.`);

En ES5, manejamos las nuevas líneas de la siguiente manera:


var text = (
    'cat\n' +
    'dog\n' +
    'nickelodeon'
);

O:


var text = [
    'cat',
    'dog',
    'nickelodeon'
].join('\n');

Los Template Literals preservarán nuevas líneas para nosotros sin tener que colocar de forma explícita en:


let text = ( `cat
dog
nickelodeon`
);

En los Template Literals pueden aceptar expresiones, así:


let today = new Date();
let text = `The time and date is ${today.toLocaleString()}`;

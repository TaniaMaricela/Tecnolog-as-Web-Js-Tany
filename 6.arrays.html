Destructuring arrays
arreglos

La Desestructuración nos permite extraer los valores de las matrices y los objetos (incluso muy anidadas) y almacenarlos en las variables con una sintaxis más conveniente.

Destructuring Arrays
desestructuración de matrices 


var arr = [1, 2, 3, 4];
var a = arr[0];
var b = arr[1];
var c = arr[2];
var d = arr[3];
let [a, b, c, d] = [1, 2, 3, 4];

console.log(a); // 1
console.log(b); // 2
